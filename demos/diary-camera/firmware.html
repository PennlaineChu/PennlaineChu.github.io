LED State Machine Reference

Phase Trigger LED Behavior
Standby/Boot Power on Pulse blue → Solid green
Ready to Shoot Short press shutter Solid white (lens active)
Capture Short press again Flash white (photo taken)
Diary Recording Long-press <3s after snap Rainbow pulse (recording)
Processing Release button Spinning red (uplink/AI)
Success AI response ready Solid green (return to standby)

Default path: If no long-press within 3s of snap → silent photo, return to standby.

Tech Stack

  • Framework: Arduino ESP32-S3 core
  • LED Control: FastLED or Adafruit NeoPixel (WS2812 via RMT)
  • Audio: I2S for both mic and speaker (reduces Wi-Fi EMI)
  • Connectivity: Wi-Fi HTTP POST to Flask backend
  • Storage: SD card for photos, PSRAM for staging
