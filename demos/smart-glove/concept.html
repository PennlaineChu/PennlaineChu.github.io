Wearable В· Presbyopia В· Gesture-first

Smart Magnifying Glove

End-to-end concept: fingertip camera, wrist MCU + OLED, flex and IMU inputвҖ”magnify small text without pulling out a phone.

Prototype 1 lab Checklist, BOM, diary, photos

Problem

Presbyopia makes near text hard to read. A phone works, but it breaks flowвҖ”unlock, frame, zoom. The target is a self-contained glove: glance at the wrist, pinch to grab a readable view, stay hands-on with the page or part.

Vision

Goals

  • Pinch to capture & magnify
  • Digital zoom 2Г— вҶ’ 4Г— вҶ’ 8Г—
  • Gesture-only UX on the device
  • Fully wearable stack

Prototype 1 scope

First milestone is a bench proof: ESP32-S3, OV2640, ST7789 OLEDвҖ”stable camera-to-display path and sane latency. Flex, IMU, glove integration, and polish follow in later phases (tracked in the lab).

Roadmap

Five phases from procurement through software polish. Task checkboxes and dates sit in the lab.

Phase 0

Planning & procurement В· week 0

Research, order parts, bench toolchain.

Phase 1

Proof of concept В· weeks 1вҖ“2

Camera вҶ’ OLED pipeline on the dev board; frame rate and latency baseline.

Phase 2

Sensor integration В· weeks 3вҖ“4

Flex ADC, smoothing, calibration, gesture state machine.

Phase 3

Wearable integration В· weeks 5вҖ“8

PCB, glove, TPU housing, harnesses, on-body assembly.

Phase 4

Software & tuning В· weeks 9вҖ“10

Zoom, focus peaking, power modes, optional OCRвҖ”target ~15 fps preview, low capture latency.

After prototype 1

  • Haptic feedback for gesture confirmation
  • Bluetooth to phone for saving images
  • Voice commands
  • High-contrast / night display modes
  • On-device text detection (ML)
