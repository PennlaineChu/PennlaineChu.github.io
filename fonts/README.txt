Drop the Britney font file here.

Accepted filenames (any one works; .woff2 preferred for speed):
  Britney-Regular.woff2
  Britney-Regular.woff
  Britney-Regular.otf
  Britney-Regular.ttf

Once the file is in this folder, all display/script headings on the site
switch to Britney automatically — no code changes needed.

Tip: if you only have a .ttf/.otf, you can convert it to .woff2 at
https://cloudconvert.com/ttf-to-woff2 for faster loading.
